HTML

  • HTML is used to build the structure of a webpage and provide a framework that makes it user-friendly. Think of HTML like the frame of a house.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element defines the title for a webpage and is extremely important for SEO.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.

CSS

  • CSS stands for Cascading Style Sheets and defines how webpage content is presented to users.
  • Inline CSS, Internal CSS Style Sheet and External CSS Style sheets are three ways to style a webpage.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git branch: lists your branches
  • git checkout: -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is a high-level dynamic programming language that allows users to interact with websites.
  • Control flow is the order in which a computer executes code in a script.
  • A variable is a named container that allows us to store data in our code.
  • An array is a single variable that is used to hold data that is related in some way.
  • Remember that arrays are zero indexed and sequential - that means 0 is the first number.
  • In a for loop, the first statement determines the starting point of loop, the second statement is the condition, and the final statements allows the array to iterate over each item.
  • Functions are reusable blocks of code that contain instructions to perform a specific task.
